## 🏗️ Data Engineering – Building the Data Warehouse

### 🎯 Objective
Develop a modern data warehouse using SQL Server to consolidate sales data from CRM and ERP systems, enabling analytical reporting and informed decision-making.

### ⚙️ Specifications
- **Data Sources:** Two CSV files from ERP and CRM systems.
- **Data Quality:** Cleanse and resolve data quality issues prior to analysis.
- **Integration:** Merge both sources into a single, user-friendly data model.
- **Scope:** Focused on the latest dataset only (no historization).
- **Documentation:** Clear data model structure to support business and analytics use.

---

## 📈 Data Analysis – BI and Reporting

### 🎯 Objective
Deliver SQL-based analytics to uncover key business insights:

- 🧍‍♂️ **Customer Behavior**
- 📦 **Product Performance**
- 💰 **Sales Trends**

These insights help business stakeholders make strategic decisions based on real data.
